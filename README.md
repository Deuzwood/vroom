# Vroom

Multiplayer cars game with  [Socket.io](http://socket.io/) and [Three.js](http://threejs.org/)

Herokuapp ready

### Prerequisites
- npm
- nodejs

## Install

Installation des modules avec **npm**

```
npm install
```

## Usage

```
npm run start
```
With nodemon : 
```
npm run dev
```
